🌐 翻译 API 核心配置
填写翻译服务 API Key。推荐 OpenRouter 或 DeepSeek（DeepSeek 默认模型 deepseek-v4-flash）。
🧠 DeepSeek 思考模式
启用后请求体携带 {"thinking":{"type":"enabled"}}
• 同一模型同时支持思考/非思考（由 API 参数控制，无需换模型）
• 非思考模式：自动传 {"type":"disabled"}（DeepSeek 默认开启思考）
• 思考模式下 temperature 等参数不生效，将自动忽略
• 多轮无工具调用：reasoning_content 可选传回（API 忽略）
• 多轮有工具调用：reasoning_content 必须传回（否则 400 错误）
